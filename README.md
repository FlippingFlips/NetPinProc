## NetPinProc
---

Dotnet standard2.0 wrapper implementation for the `libpinproc` ProcDevice. For interfacing with a P-ROC or P3-ROC controller boards over USB.

This resides in it's own library because it depends on the `linpinproc` library. Libpinproc builds are included and can be used from the `lib/x64` or `lib/x86`.

- Windows, Mac, Linux included.

https://github.com/preble/libpinproc
https://github.com/FlippingFlips/libpinproc
